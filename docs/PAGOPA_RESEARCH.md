## Modalità di Integrazione PagoPA per Enti Creditori

PagoPA mette a disposizione due modalità di integrazione:

### 1. Integrazione tramite API asincrone
- Modalità consigliata per la maggior parte degli Enti
- Non richiede conferimento obbligatorio all'ACA

### 2. Integrazione tramite API sincrone
- Richiede il conferimento di tutte le posizioni debitorie all'Archivio Centralizzato Avviso (ACA)
- Obbligatorio per chi sceglie questa modalità

### Casi d'uso principali
- Pagamento di un avviso presso PSP
- Pagamento spontaneo presso PSP
- Pagamento presso frontend dell'EC
- Pagamento da Touchpoint PagoPA

### Funzionalità per Ente Creditore
- Adesione

- Attestazione di pagamento

- Stampa avvisi pagoPA
- Processo di avvio in Esercizio
